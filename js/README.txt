SPACE RUNNER PRO PWA

How to use:
1. Keep the folder structure exactly as provided.
2. Host the project on localhost or HTTPS.
3. Open it in Safari on iPhone.
4. Tap Share > Add to Home Screen.
5. Launch from your Home Screen like an app.

Controls:
- Jump button = Jump
- Shoot button = Shoot
- Pause button = Pause
- Music button = Toggle music

Desktop controls:
- Space / Arrow Up = Jump
- F = Shoot
- P = Pause
- R = Restart after game over

Notes:
- PWA requires localhost or HTTPS.
- Opening index.html directly with file:// will not install as a PWA.
- Best tested using Safari on iPhone.

Suggested local server:
python -m http.server 8000

Then open:
http://YOUR_COMPUTER_IP:8000
